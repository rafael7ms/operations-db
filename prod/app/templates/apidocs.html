API Documentation

Base URL: http://127.0.0.1:5443/api

Health Check

GET /health

Returns health status of the API.

curl http://127.0.0.1:5443/api/health
Response:
{"status": "healthy"}

Employees

GET /employees

Get all employees (filter by status with query param).

# Get all active employees curl http://127.0.0.1:5443/api/employees # Get employees with status filter curl http://127.0.0.1:5443/api/employees?status=Active
GET /employees/<id>

Get single employee by ID.

curl http://127.0.0.1:5443/api/employees/41031748911
POST /employees

Create new employee.

curl -X POST http://127.0.0.1:5443/api/employees \ -H "Content-Type: application/json" \ -d '{ "employee_id": 41031748911, "first_name": "John", "last_name": "Doe", "company_email": "john@company.com", "batch": "2024-01", "supervisor": "Jane Smith", "manager": "Bob Johnson", "shift": "Morning", "department": "Customer Support", "role": "Associate" }'
PUT /employees/<id>

Update employee.

curl -X PUT http://127.0.0.1:5443/api/employees/41031748911 \ -H "Content-Type: application/json" \ -d '{ "tier": 2, "status": "Active" }'
DELETE /employees/<id>

Delete employee.

curl -X DELETE http://127.0.0.1:5443/api/employees/41031748911
POST /employees/batch

Bulk import employees from Excel file.

curl -X POST http://127.0.0.1:5443/api/employees/batch \ -F "file=@employees.xlsx"

Excel columns: Odoo ID, First Name, Last Name, Batch, Supervisor, Manager, Shift, Department, Role, Hire Date, Company Email, Tier (optional)

Schedules

GET /schedules

Get schedules with optional filters.

curl http://127.0.0.1:5443/api/schedules?start_date=2026-02-01&end_date=2026-02-28
POST /schedules

Create single schedule.

curl -X POST http://127.0.0.1:5443/api/schedules \ -H "Content-Type: application/json" \ -d '{ "employee_id": 41031748911, "start_date": "2026-02-17", "start_time": "08:00", "stop_date": "2026-02-17", "stop_time": "17:00", "work_code": "REG" }'
POST /schedules/batch

Bulk import schedules with RUEX ID matching.

curl -X POST http://127.0.0.1:5443/api/schedules/batch \ -F "file=@schedules.xlsx" \ -F "employee_file=@employees.xlsx"

RUEX ID: First letter + last name (e.g., "John Smith" â†’ "jsmith")

Attendances

GET /attendances

Get attendance records with optional filters.

curl http://127.0.0.1:5443/api/attendances?start_date=2026-02-01&end_date=2026-02-28
POST /attendances

Create attendance record.

curl -X POST http://127.0.0.1:5443/api/attendances \ -H "Content-Type: application/json" \ -d '{ "employee_id": 41031748911, "date": "2026-02-17", "check_in": "08:00", "check_out": "17:00", "exception_type": "OT", "notes": "Overtime" }'
PUT /attendances/<emp_id>/<date>

Update attendance record.

curl -X PUT http://127.0.0.1:5443/api/attendances/41031748911/2026-02-17 \ -H "Content-Type: application/json" \ -d '{"check_out": "18:00"}'
POST /attendances/batch

Bulk import attendance from Excel.

curl -X POST http://127.0.0.1:5443/api/attendances/batch \ -F "file=@attendances.xlsx"

Rewards

GET /rewards/reasons

Get active reward reasons.

curl http://127.0.0.1:5443/api/rewards/reasons
GET /rewards/employee/<id>

Get employee reward history.

curl http://127.0.0.1:5443/api/rewards/employee/41031748911
GET /rewards/employee/<id>/balance

Get employee's current point balance.

curl http://127.0.0.1:5443/api/rewards/employee/41031748911/balance Response: {"employee_id": 41031748911, "employee_name": "Heily Melendez", "point_balance": 500}
POST /rewards/award

Award points to employee.

curl -X POST http://127.0.0.1:5443/api/rewards/award \ -H "Content-Type: application/json" \ -d '{ "employee_id": 41031748911, "reason_id": 1, "points": 50, "date_awarded": "2026-02-16", "notes": "Excellent performance" }'
POST /rewards/award/batch

Bulk award points from Excel.

curl -X POST http://127.0.0.1:5443/api/rewards/award/batch \ -F "file=@rewards.xlsx"

Excel columns: Employee - ID, Reason ID, Points, Date Awarded

POST /rewards/redemptions

Redeem points for employee.

curl -X POST http://127.0.0.1:5443/api/rewards/redemptions \ -H "Content-Type: application/json" \ -d '{ "employee_id": 41031748911, "points_redeemed": 100, "redemption_type": "Gift Card", "redemption_details": "Amazon $50", "approved_by": 41000000001 }'

Exceptions

GET /exceptions

Get exception records.

curl http://127.0.0.1:5443/api/exceptions?status=Pending
POST /exceptions

Create exception record.

curl -X POST http://127.0.0.1:5443/api/exceptions \ -H "Content-Type: application/json" \ -d '{ "employee_id": 41031748911, "exception_type": "Training", "start_date": "2026-02-20", "end_date": "2026-02-22", "work_code": "TRAIN", "status": "Pending" }'
POST /exceptions/<id>/process

Process exception record.

curl -X POST http://127.0.0.1:5443/api/exceptions/1/process \ -H "Content-Type: application/json" \ -d '{"processed_by": 41000000001}'
POST /exceptions/batch

Bulk import exceptions from Excel.

curl -X POST http://127.0.0.1:5443/api/exceptions/batch \ -F "file=@exceptions.xlsx"

Admin Options

GET /admin/options

Get admin dropdown options.

curl http://127.0.0.1:5443/api/admin/options?category=department
POST /admin/options

Create admin option.

curl -X POST http://127.0.0.1:5443/api/admin/options \ -H "Content-Type: application/json" \ -d '{"category": "department", "value": "Customer Support"}'
